# Advent of Code 2025 in Gleam

Developed using Gleam 1.13.0.

## Usage

You have to supply the puzzle input files yourself.
Place them inside the `inputs` folder and name them like `day01.txt`.

```
Usage: gleam run day part
Example: gleam run 1 2
```

## Development

```sh
gleam run   # Run the project
gleam test  # Run the tests
```
